# Package index

## All functions

- [`bccamtrapp()`](https://bcgov.github.io/bccamtrap/reference/bccamtrapp.md)
  : Launch Shiny app

- [`fill_spi_template()`](https://bcgov.github.io/bccamtrap/reference/fill_spi_template.md)
  : Write all data to a new SPI submission file

- [`fill_spi_template_ff()`](https://bcgov.github.io/bccamtrap/reference/fill_spi_template_ff.md)
  : Write data to a new SPI submission file from field form data

- [`make_deployments()`](https://bcgov.github.io/bccamtrap/reference/make_deployments.md)
  : Create a spatial data frame of deployments

- [`make_sample_sessions()`](https://bcgov.github.io/bccamtrap/reference/make_sample_sessions.md)
  : Identify distinct sample sessions from timelapse image data

- [`map_stations()`](https://bcgov.github.io/bccamtrap/reference/map_stations.md)
  : Visualize stations on an interactive map

- [`merge_deployments_images()`](https://bcgov.github.io/bccamtrap/reference/merge_deployments_images.md)
  : Merge image data with deployment data

- [`plot_deployment_detections()`](https://bcgov.github.io/bccamtrap/reference/plot_deployment_detections.md)
  : Plot image timestamps over deployment periods

- [`plot_deployments()`](https://bcgov.github.io/bccamtrap/reference/plot_deployments.md)
  : Plot timelines of image deployments

- [`plot_diel_activity()`](https://bcgov.github.io/bccamtrap/reference/plot_diel_activity.md)
  : Plot daily activity of species detected by motion detection

- [`plot_snow()`](https://bcgov.github.io/bccamtrap/reference/plot_snow.md)
  : Plot snow depths across deployments

- [`qa_deployment_images()`](https://bcgov.github.io/bccamtrap/reference/qa_deployment_images.md)
  : Check for mismatched deployment labels in deployment data and image
  data

- [`qa_image_data()`](https://bcgov.github.io/bccamtrap/reference/qa_image_data.md)
  : Perform a series of QA checks on the Timelapse Image data

- [`qa_stations_spatial()`](https://bcgov.github.io/bccamtrap/reference/qa_stations_spatial.md)
  : Check spatial attributes of Sample Station information

- [`rai_by_time()`](https://bcgov.github.io/bccamtrap/reference/rai_by_time.md)
  : Calculate RAI over a window of time, optionally rolling

- [`read_cam_setup_checks()`](https://bcgov.github.io/bccamtrap/reference/read_cam_setup_checks.md)
  : Read the "Camera Setup and Checks" tab from the RISC worksheet.

- [`read_camera_info()`](https://bcgov.github.io/bccamtrap/reference/read_camera_info.md)
  : Read the "Camera Information" tab from the RISC worksheet.

- [`read_deployments_csv()`](https://bcgov.github.io/bccamtrap/reference/read_deployments_csv.md)
  : Read csv output from deployments field form

- [`read_image_data()`](https://bcgov.github.io/bccamtrap/reference/read_image_data.md)
  : Read image data from a collection of csvs from TimeLapse

- [`read_project_info()`](https://bcgov.github.io/bccamtrap/reference/read_project_info.md)
  : Read project-level data from the RISC worksheet 'Project
  Information' tab

- [`read_sample_station_csv()`](https://bcgov.github.io/bccamtrap/reference/read_sample_station_csv.md)
  : Read csv output from sample station field form

- [`read_sample_station_info()`](https://bcgov.github.io/bccamtrap/reference/read_sample_station_info.md)
  : Read the "Sample Station Information" tab from the RISC worksheet.

- [`sample_rai()`](https://bcgov.github.io/bccamtrap/reference/sample_rai.md)
  : Calculate Relative Activity Index (RAI)

- [`to_sf()`](https://bcgov.github.io/bccamtrap/reference/to_sf.md) :

  Convert Sample Station Info or Camera Info to a spatial `sf` boject
  \#'

- [`write_image_data()`](https://bcgov.github.io/bccamtrap/reference/write_image_data.md)
  : Write image data to a csv file

- [`write_to_spi_sheet()`](https://bcgov.github.io/bccamtrap/reference/write_to_spi_sheet.md)
  : Write a single data frame to SPI submission template
